# PROJECT_CONTEXT.md

## 🎯 Общее описание проекта
*   **Название бота:** DJgurda Bot
*   **Основная цель:** Telegram бот для обработки медиа-ссылок из групповых чатов. Преобразует ссылки на контент из различных платформ (TikTok, YouTube, Instagram, Яндекс.Музыка) в унифицированный формат, доступный всем участникам беседы.
*   **Статус проекта:** Active Development
*   **Целевая аудитория:** Пользователи Telegram групповых чатов, где участники имеют доступ к разным медиа-платформам.

## 🛠 Технологический стек
**Язык программирования:** Python 3.11
**Основные зависимости:**
- `aiogram>=3.25.0` - асинхронный фреймворк для Telegram Bot API
- `sqlalchemy>=1.4` + `aiosqlite>=0.22.0` - асинхронная работа с БД
- `yt-dlp` - загрузка видео/аудио с YouTube и других платформ
- `yandex-music==2.2.0` - работа с Яндекс.Музыкой
- `beautifulsoup4>=4.14.0` - парсинг HTML

**Инфраструктура:**
- **Контейнеризация:** Docker + официальный python:3.11-slim образ
- **Медиа обработка:** FFmpeg предустановлен в контейнере
- **База данных:** SQLite с асинхронным драйвером
- **CI/CD:** GitHub Actions с автоматическим деплоем на VPS

**Среда выполнения:** 
- Разработка: локально
- Продакшен: VPS с Docker (две среды: dev и prod)

## 💾 Архитектура обработчиков

**Базовые компоненты:**
- `BaseHandler` - абстрактный класс для всех обработчиков
- `ServiceManager` - регистрация и поиск обработчиков по URL
- Миксины (`VideoMixin`, `PhotoMixin`, `AudioMixin`, `MediaGroupMixin`) - общая функциональность

**Поддерживаемые типы контента:**
- Видео (с миниатюрами)
- Аудио (с обложками)
- Фото и группы медиа (слайдшоу)
- Профили пользователей

## 💾 Система команд бота

**Основные команды:**
- `/help` - список поддерживаемых медиа-платформ
- `/info` - информация о версии и времени работы бота
- `/statistics` - статистика активности в чате (топ пользователей)

**Команды управления:**
- `/start`, `/stop`, `/toggle_bot` - включение/отключение бота в чате
- `/enable_errors`, `/disable_errors`, `/toggle_errors` - управление показом ошибок
- `/enable_notifications`, `/disable_notifications`, `/toggle_notifications` - управление уведомлениями

## 💾 Данные и база данных

**Основные сущности:**
- Настройки бота (bot_settings) - состояние бота в чатах
- Статистика (stats) - активность пользователей
- Источники (sources) - поддерживаемые медиа-платформы

**Ограничения:**
- `MAX_CAPTION = 1024` - максимальная длина подписи
- `MAX_AGE_SECONDS = 3600` - время жизни кэша
- Лимиты размеров файлов: фото 10MB, видео/аудио 50MB

## 🔧 Конфигурация и переменные окружения

**Обязательные переменные:**
- `BOT_TOKEN` - токен Telegram бота
- `BOT_DB_PATH` - путь к базе данных
- `YANDEX_MUSIC_TOKEN` - токен Яндекс.Музыки
